ความเป็นส่วนตัว

ชี้แจงการเก็บข้อมูลในส่วนต่างๆ และชี้แจงการใช้งานโดยละเอียด

ประวัติการสนทนา

ทั้งมูมู่และไอริน มีการเก็บประวัติสนทนาแบบไม่ระบุผู้พูดไว้บนเซิฟเวอร์เพื่อพัฒนาให้ระบบสามารถเรียนรู้ได้ดีขึ้นสำหรับคำที่ไม่รู้จัก โดยที่ มูมู่นั้นจะมีการนำประวัติสนทนาไปแสดงยัง https://irin.in.th/mumu ด้วย จึงอยากให้ผู้ใช้งานพึงระวังข้อมูลที่เป็นความลับส่วนบุคไว้ด้วย หากพบว่ามีข้อมูลส่วนบุคคลหลุดรั่วออกขึ้นมายังเซิฟเวอร์ ท่านสามารถแจ้งลบกับเราได้ทันที แต่เราจะไม่รับผิดชอบต่อความเสียหายใดๆก็ตามที่เกิดขึ้นทั้งสิ้น

การสอนศัพท์ใหม่

สำหรับมูมู่ การสอนศัพท์ใหม่นั้นเราจำเป็นจะต้องเก็บชื่อผู้ใช้งานเฟสบุ๊คไว้บนเซิฟเวอร์เพื่ออ้างถึงคนสอน เพื่อเป็นมาตราการความปลอดภัยให้แก่ตัวมูมู่ และตัวผู้ใช้เอง ซึ่งเราจะไม่เปิดเผยชื่อผู้สอนให้แก่ผู้ใช้ ยกเว้น จะมีผู้ใช้ร้องขอ และทางผู้ดูแลเห็นสมควร แต่ผู้ดูแลจะไม่รับผิดชอบใดๆเกี่ยวกับความเสียหายที่เกิดขึ้นจากคำศัพท์ที่สอนหรือการร้องขอดูชื่อผู้สอนทั้งสิ้น สำหรับไอรินท่านสามารถส่ง pull request เข้ามายัง repository ของไอรินได้ แต่เมื่อทางทีมงานไอรินได้ทำการรวมโค้ดเข้ามายัง repository ของไอรินแล้ว ความเสียหายใดๆที่เกิดขึ้นนั้นเป็นความผิดของตัวผู้ส่ง pull request ไม่เกี่ยวข้องกับผู้ดูแลไอรินไม่ว่าในกรณีใดๆทั้งสิ้น
