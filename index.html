ไอริน ระบบตอบโต้อัตโนมัติ

พัฒนาขึ้นจากโค้ดนับพันบรรทัดสู่การโต้ตอบอัจฉริยะ
สามารถประยุกต์ให้เข้ากับงานของคุณได้อย่างง่ายดาย

Next

ประยุกต์ใช้งานได้หลากหลาย

ไอรินถูกพัฒนาขึ้นมาเพื่อถามตอบคำถามกับมนุษย์โดยถูกปรับแต่งเป็นพิเศษเพื่อให้ใช้งานกับภาษาไทยได้

คุณสามารถพบไอรินได้ในทุกที่ไม่ว่าจะเป็นอุปกรณ์โทรศพท์มือถือ ตู้เย็น หรือแม้กระทั่งหม้อหุงข้าว เพื่อตอบสนองกับผู้ใช้งาน ได้ทันที โดยไม่ต้องรอเจ้าหน้าที่ตอบ ช่วยทำให้ผู้ใช้มีประสบการณ์ที่ดีขึ้นในการใช้งานไม่ต้องรอคอยเจ้าหน้าที่เป็นเวลานาน ซึ่งทางโปรแกรมเมอร์สามารถเขียนคำสั่งเพื่อตรวจสอบถ้อยคำที่ไอรินตอบ จึงสามารถให้ระบบของคุณทำงานตามการตอบของไอรินได้

และไอรินยังเปิดกว้างสำหรับโปรแกรมเมอร์ โดยสามารถดาวน์โหลด ซอร์สโค้ดบางส่วนของไอรินไปพัฒนาต่อได้ทันที พร้อมทั้งทางเว็บไอรินเองมีบริการ Rest API และคุณสามารถสร้างบอทเองได้ด้วยตัวเองผ่านทางเว็บไอริน ทำให้ไอรินสามารถเข้าไปอยู่ได้ทกุกอุปกรณ์ เนื่องจากความง่ายในการพัฒนา

สร้างบอทใช้เองไม่ใช่เรื่องยาก

ด้วยการใช้ตัวจัดการบอทของไอรินคุณสามารถทำมันได้โดยไม่จำเป็นต้องเขียนโปรแกรม

เราได้สร้างตัวจัดการบอทขึ้นมาให้ใช้งานง่าย คุณไม่จำเป็นต้องสัมผัสโค้ดเพื่อสร้างบอทของคุณเลยแม้สักนิดเดียว ช่วยให้คุณสามารถสร้างบอทของคุณได้อย่างรวดเร็ว นอกจากนี้ยังง่ายต่อการควบคุมและแบ่งปันไปให้ผู้อื่นใช้งานด้วย

เปิดกว้างเสมอสำหรับโปรแกรมเมอร์

สามารถนำไอรินไปติดตั้งบนเซิร์ฟเวอร์หรืออุปกรณ์ของตนเองได้อย่างง่ายดาย

ไอรินถูกสร้างขึ้นด้วย Rivescript ซึ่งมี Interpeter เป็น Javascript ทำให้สามารถใช้ได้บน Node.JS และ Webbrowser ซึ่งทำให้สามารถใช้งานได้บนเกือบทุกอุปกรณ์ จึงไม่ใช่เรื่องยากที่จะใช้ไอรินบนอุปกรณ์ของคุณ และไอรินยังมีบริการ Rest API บนเว็บของไอรินเอง ทำให้โปรแกรมเมอร์ไม่จำเป็นต้องมีความรู้ด้านการใช้ Javascript ก็สามารถดึงข้อมูลของไอริน ผ่านที่นี่เพื่อนำไอรินไปใช้ยังแพลตฟอร์มอื่น

และทั้งหมดนี่คือไอริน

จากทั้งหมดที่กล่าวมาสามารถสรุปเป็นความสามารถของไอรินได้ดังนี้

ใช้งานง่าย

ไม่จำเป็นต้องเขียนโปรแกรมได้ เพียงแค่ใช้ตัวจัดการบอทของไอรินก็ทำให้คุณก็รู้สึกสนุกกับการสร้างบอทของตนเองได้แล้ว

รองรับภาษาไทยและอังกฤษ

ตัวจัดการบอทของไอรินรองรับทั้งภาษาไทยและภาษาอังกฤษ คุณจึงสามารถสอนบอทให้ตอบทั้งภาษาไทยและภาษาอังกฤษได้

ง่ายต่อการแบ่งปัน

เมื่อคุณสร้างบอทของตัวเองเสร็จ คุณสามารถส่งลิงก์ บอทของคุณให้เพื่อนของคุณสามารถเข้ามาสนทนาโต้ตอบกับบอทของคุณได้

ใช้งานได้ทุกที่

บอทที่สร้างขึ้น สามารถใช้งานได้ทุกที่ ด้วยการติดตั้ง Rivescript Interpeter หรือใช้ Rest API ในการประมวลผล

ทำตามที่คุณสั่ง

บอทจะเรียนรู้จากที่คุณสอนและจะเป็นในแบบที่คุณต้องการให้เป็น ตอบโจทย์ความต้องการใช้งานบอทของคุณ

ใช้งานได้ฟรี

ไอรินเปิดให้ขั้นฐานได้ฟรี เราขอเพียงคุณบริจาคเงินเล็กน้อยเท่านั้น เพื่อใช้ในการบริหารค่าใช้จ่ายต่างๆของเซิฟเวอร์
